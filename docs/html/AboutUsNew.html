Sobre el Proyecto

Diseño de página web de E-commerce para el proyecto “Somos Booking” enfocado en la promoción de servicios para bandas emergentes. El objetivo de la página web es poder dar a conocer los servicios con los que cuenta el proyecto “Somos booking” para poder llegar a más colaboradores y hacer crecer la “Familia Somos Booking”. También se pretende exponer los eventos pasados mediante flyers y galería de fotos y comercializar los boletos en preventa para los próximos a realizarse; así mismo, se estaría agregando a la tienda la merch de las bandas con las que se colabora actualmente.
